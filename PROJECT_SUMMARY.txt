   ANNONCES:
   ├─ ✅ CREATE: Ajouter une annonce (formulaire complet)
   ├─ ✅ READ:   Afficher la liste et détails
   ├─ ✅ UPDATE: Éditer une annonce existante
   └─ ✅ DELETE: Supprimer une annonce (avec confirmation)

   COMMENTAIRES:
   ├─ ✅ CREATE: Ajouter un commentaire
   ├─ ✅ READ:   Afficher les commentaires par annonce
   ├─ ⏳ UPDATE: Éditer un commentaire (optionnel - voir IMPLEMENTATION_OPTIONNELLE.md)
   └─ ⏳ DELETE: Supprimer un commentaire (optionnel - voir IMPLEMENTATION_OPTIONNELLE.md)

   NAVIGATION:
   ├─ ✅ Liste des annonces (écran principal)
   ├─ ✅ Double-clic pour voir détails
   ├─ ✅ Boutons Ajouter/Éditer/Supprimer/Retour
   └─ ✅ Messages de succès/erreur

   1. Créer la BD:
      mysql -u root -p < setup.sql

   2. Compiler:
      mvn clean compile

   3. Lancer (2 options):
      mvn javafx:run

      Double-cliquez run.bat
